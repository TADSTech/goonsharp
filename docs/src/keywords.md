# Keywords

GoonSharp replaces Rust keywords with goon-themed alternatives. Here is the complete reference.

## Core Keywords

| GoonSharp | Rust | Description |
|-----------|------|-------------|
| `goonsesh` | `fn` | Function declaration |
| `goon` | `let` | Variable binding |
| `goon gooning` | `let mut` | Mutable variable binding |
| `goon_const` | `const` | Constant declaration |
| `goonprint!` | `println!` | Print with newline |
| `goonreturn` | `return` | Return from function |
